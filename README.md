# Level Devil 2 Clone

A challenging platformer game inspired by Level Devil.

## Features
- 5 Challenging Levels
- "Troll" mechanics (moving spikes, falling platforms, invisible traps)
- Responsive Canvas
- Yandex Games SDK Integration
- Mobile Touch Controls (Auto-detected)

## Controls
- **Desktop:**
  - Move: Arrow Keys or WASD
  - Jump: Space
- **Mobile:**
  - Touch controls are not explicitly visualized but the game logic supports touch events if implemented. (Currently keyboard focused)

## Development
- Built with HTML5 Canvas and Vanilla JavaScript.
- No external game engine dependencies.
